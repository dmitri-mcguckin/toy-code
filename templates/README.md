# Circularly Linked List

This code is one way to make use of templates in C++ using tools provided by C++ 14 and a few misc 3rd party libraries.

## Development and Deployment

### Compilation

Make  new:

`$` `make`

Make clean:

`$` `make clean`

Make and run:

`$` `make run`

Make and check for memory leaks:

`$` `make leak`
